## 🔴 高优先级功能

### 1. 粉丝/关注系统

**优先级**: 🔴 高  
**预估工作量**: 后端 2-3天 + 前端 3-4天  
**相关文件**: 
- 后端：`controller/follow.go`、`logic/follow.go`、`dao/mysql/follow.go`
- 前端：`components/FollowButton.vue`、`views/AuthorPage.vue`、`stores/follow.ts`

**功能描述**:
- 用户可以关注作者
- 仪表板显示粉丝数
- 作者主页显示粉丝列表
- 前端粉丝卡片不再显示"敬请期待"

**后端 API 需求**:
- `POST /follow` - 关注用户
- `DELETE /follow` - 取消关注
- `GET /followers/:user_id` - 获取粉丝列表
- `GET /following/:user_id` - 获取关注列表（可选）

**前端页面**:
- 作者主页显示"关注"按钮
- 点击关注后显示"已关注"并可取消
- 显示粉丝数
