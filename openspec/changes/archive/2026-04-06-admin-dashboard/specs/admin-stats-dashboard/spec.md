## 新增需求

### 需求:展示全站数据概览卡片
管理员必须能在统计页看到全站核心指标的汇总卡片。

#### 场景:加载概览数据
- **当** 管理员访问系统统计页
- **那么** 必须调用 `GET /api/v1/admin/stats/overview` 并展示：总用户数、总文章数、总评论数、总点赞数

### 需求:展示每日新增趋势图
管理员必须能看到最近 30 天内用户、文章、评论的每日新增数量趋势图。

#### 场景:加载趋势图
- **当** 管理员访问系统统计页
- **那么** 必须调用 `GET /api/v1/admin/stats/daily`（默认最近 30 天）并用折线图展示每日新增趋势

#### 场景:图表包含三条折线
- **当** 趋势图渲染完成
- **那么** 必须同时显示用户新增、文章新增、评论新增三条折线，并有图例区分
