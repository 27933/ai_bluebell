## ADDED Requirements

### ADMIN_TEST-001: 管理员获取文章列表接口必须正确工作
GET /api/v1/admin/articles 接口必须允许管理员查看所有文章。

#### 场景:管理员获取所有文章
- **当** 管理员发送获取文章列表请求
- **那么** 系统必须返回所有文章（包括草稿、下线状态）
- **并且** 支持按状态筛选

#### 场景:普通用户访问管理员接口
- **当** 非管理员用户发送请求
- **那么** 系统必须返回 HTTP 403 状态码

#### 场景:未登录用户访问管理员接口
- **当** 未携带 Token 发送请求
- **那么** 系统必须返回 HTTP 401 状态码

### ADMIN_TEST-002: 管理员设置文章精选接口必须正确工作
PATCH /api/v1/admin/articles/:id/featured 接口必须允许管理员设置文章精选状态。

#### 场景:管理员设置文章精选
- **当** 管理员发送 is_featured=true 的请求
- **那么** 系统必须更新文章精选状态

#### 场景:管理员取消文章精选
- **当** 管理员发送 is_featured=false 的请求
- **那么** 系统必须取消文章精选状态
