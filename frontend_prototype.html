什么是 Composition API？

Vue 3 引入的 Composition API 是一种新的组件编写方式，相对于 Options API，它提供了更好的代码组织和重用能力。

核心概念

Composition API 围绕 setup 函数展开，在这个函数中你可以使用各种 API 来定义组件的逻辑。

import { ref, computed, watch } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const doubled = computed(() => count.value * 2)

    watch(count, (newVal) => {
      console.log(`Count changed to ${newVal}`)
    })

    return { count, doubled }
  }
}

最佳实践

1. 逻辑分离 - 将相关的逻辑聚集在一起
2. 代码重用 - 使用 Composable 函数提高代码复用率
3. 类型安全 - 充分利用 TypeScript 的类型系统
4. 性能优化 - 合理使用 computed 和 watch
